## Quick start

```python
from tikzpics import TikzFigure, Node

fig = TikzFigure()
a = fig.add_node(Node(0, 0, content="A"))
b = fig.add_node(Node(2, 0, content="B"))
fig.draw(a, b, arrow="->")
fig.show()
```
